I've been fascinated with the internet since I was a kid and am excited to be contributing some content and work of my own. rusticCode is a one man show, my online "brand" if you will, which simply represents the juxtaposition of an old-school work ethic colliding with the technology of today. Modern, Flat Login Template

Here's a nifty little login form I built after being inspired by some of the financial/banking websites I use - I noticed went through a "refresh" of sorts in the past year or so, updating their overall user experience. This is my take on what's popular in user interface design at the moment. Feel free to use and modify this template, keep in mind it has not been built with any security features, reg-ex validation for email, etc... get the code here

Passport 865 ESO Grid

This is another contribution I made while working on the Passport 865 project, dubbed the "Passport 865 ESO Grid" - this infographic was built to provide entrepreneurs with a simple visual tool to lead them to the appropriate ESO(Entrepreneur Service Organizations) based on the current stage of their business or start-up. I built this using Venngage.com, which offers tools to create infographics, check out the full pdf here.

Wedding Info/RSVP

I would have to say this is the most personal project I have developed up to this point. It is a tribute as well as an announcement to my fiance' and I's upcoming wedding. It is built using w3.css repsonsive front-end framework and features a RSVP component that allows guests to enter their name and whether or not they will attend, that data is then sent to a mySQL database where guest's responses can be viewed.      

Random Quote Machine

While this app won't win any awards with its design, it does demonstrate some pretty cool things that can be achieved with plain ole JavaScript, namely writing the quotes in JSON which can then be displayed ramdonly using functions. This was a Free Code Camp challenge with the requirments being the user can click a button to show a new quote as well as click a button that will tweet out a quote.

Entertainment 865

The concept behind Entertainment 865 began in the fall of 2014 but was not developed and fully realized until fall 2015, mainly due to the simple fact that my web development skills didn't exist beyond some basic HTML and CSS at the time. With the help of skills I learned in Intro to Internet Software at PSCC, I was able to build a local entertainment site, as my final semester project, which feautured a database of celebrities from the Knoxville area and a Google calendar displaying dates and other information on concerts and events.

Passport 865

The "Entrepreneur Passport" is the result of spring 2016 Group Capstone Project at PSCC. Our task was to create a dynamic web app for Knoxville Entrepreneur Center that would allow entrepreneurs and start-ups to connect with local support organizations based on specific business needs. My primary contribution was to build a directory of resources useful to an entrepreneur or small business owner.
