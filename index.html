Recent Projects


My Record Collection

This is my final capstone project to complete my degree at Western Governors University. It is built with Ruby on Rails and uses Bootstrap on the front end. I love collecting vinyl records, so I came up with the idea of building an app that would allow users to keep track of their current record collection as well as a 'wish list', or records they want to add to their collection. One of the main features of this app is to allow users to easily move an album from their wish list to their main record collection. Another feature I added is a search bar to quickly find albums by artist, eliminating the need to scroll through the entire table.

Term Planner Mobile App

I built this app for Mobile Application Development at WGU. It is an Android app written in Java. The idea is to allow users to create as many semesters as needed and then to add courses to each semester, to keep track of the progress they are making towards earning their degree. Some of the cool features include the ability to set start and end goal dates for a course, then recieve a notification when the expected end date is approaching even if the app is not open. Also included is a "sharing" feature that lets users send course notes from the app to an email address.
